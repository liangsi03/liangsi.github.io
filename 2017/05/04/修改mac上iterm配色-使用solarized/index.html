修改mac上iterm配色(使用solarized)

发表于 2017-05-04 | 分类于 小技能

大概是我觉得原来的iterm自带的配色很丑，而且总是被吐槽，然后就强迫症发作，修改了下配色

1.下载solaized DARK
Solarized Dark 官方网站: http://ethanschoonover.com/solarized
Solarized Dark github 地址: https://github.com/altercation/solarized
2.导入上述文件

3.设置后修改bash_profile

在此基础上，可以修改PS1，这样终端会显示你想要的路径格式,我强迫自己修改成了这样，心里舒服。

4.PS1中参数说明
(1)要修改命令行提示符，我们通过环境变量PS1来达到目的。

　　命令列表的参数：

　　\u 显示当前用户的用户名（user name）

　　\h 显示主机名（host name）

　　\W 显示当前工作目录的名字（work directory）

　　\$ 显示$符作为提示符，如果用户是root的话，则显示#号

(2)文本终端的颜色可以使用“ANSI非常规字符序列”来生成

　　“\033”引导非常规字符序列

　　“m”意味着设置属性然后结束非常规字符序列

　　 编码 颜色/动作

　　0 重新设置属性到缺省设置

　　33 设置棕色前景

　　34 设置蓝色前景

　　36 设置青色前景

5.要想设置iterm进入上次打开的目录
